# VERA Platform

Embodied nervous system co-regulation sanctuary with biometric integration, haptic feedback, and immersive presence.

## Features

- **Ecosystem** - Three retreat spaces (Library, Garden, Presence) with real-time nervous system visualization
- **Laboratory** - Immersive breathing orb gateway to co-regulation space
- **Chat with VERA** - Conversational AI companion with nervous system awareness
- **Dashboard** - Real-time vagal tone, heart coherence, and nervous system state
- **Learn** - Educational resources on vagal co-regulation

## Quick Start

```bash
# Install dependencies
pnpm install

# Run dev server
cd apps/web
npm run dev

# Access at http://localhost:3000
```

## Deployment

Deployed on Vercel: https://vera-mobile.vercel.app

## Architecture

- **Framework**: Next.js 14.1 (App Router)
- **Styling**: Tailwind CSS + Custom animations
- **3D Graphics**: Three.js + React Three Fiber
- **Monorepo**: Turborepo + pnpm
